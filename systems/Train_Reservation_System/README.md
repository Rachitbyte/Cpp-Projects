This is a console-based Train Reservation System implemented in C++. It provides basic functionality for managing trains and handling seat bookings through separate admin and guest interfaces.

The system allows administrators to add new trains, view train details, and generate reports based on train type, departure, destination, and booked seats. Guests can view available trains, book seats, and check their bookings.

The project uses structured programming with arrays and structs to manage train and booking data. It demonstrates core C++ concepts such as functions, condition handling, basic data storage, and menu-driven program flow.

Key Features:

Admin login system
Add and manage train records
Search trains by type, departure, and destination
Book seats with validation
View booking details
Console-based UI with menu navigation

This project is suitable for beginners learning C++ and understanding how basic reservation systems work.
