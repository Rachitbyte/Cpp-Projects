This project demonstrates the use of aggregation and inheritance concepts in C++ through a simple college management system.

It models relationships between students and exams using classes such as Student, Exam, and College. Aggregation is used by managing collections of students and exams inside the College class, while inheritance is applied to extend exam-related functionality.

Key Concepts Covered:

Aggregation using object arrays
Inheritance (protected access)
Dynamic record management (add, view, remove)
Menu-driven console application
Basic class design and encapsulation

Features:

Add and view student records
Remove students by roll number
Add and view exam details
Structured interaction using a menu system

This project is useful for understanding how real-world relationships can be modeled using OOP principles in C++.
