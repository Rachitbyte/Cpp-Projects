This project demonstrates constructor delegation and inheritance chaining in C++ using simple course and student classes.

It shows how constructors can call other constructors within the same class (delegation) and how base class constructors are invoked in derived classes (inheritance chaining).

Key Concepts Covered:

Constructor delegation
Inheritance (base and derived classes)
Constructor chaining across classes
Default and parameterized constructors
Destructor basics

Features:

Reusable constructor logic using delegation
Inherited properties from base class
Clear demonstration of object initialization order
Simple output to verify constructor behavior

This project is focused on strengthening understanding of object initialization and class relationships in C++.
