# File Handling in C++ (Line-by-Line & Char-by-Char)

This project contains two simple C++ programs demonstrating basic file handling using the `<fstream>` library.

---

## 📂 Files Included

### 1. Line-by-Line Writing & Reading
- **File:** `FileHandling_Writing-line-by-line.cpp`
- **Description:**  
  Writes input to a file line by line using `getline()` and reads it back the same way.
- **Stop Condition:** Type `END` to stop writing.

---

### 2. Character-by-Character Writing & Reading
- **File:** `FileHandling_Writing-char-by-char.cpp`
- **Description:**  
  Writes input character by character using `getchar()` and reads it using `fin.get()`.
- **Stop Condition:** Press `*` to stop writing.

---

## ⚙️ Features

- File creation and writing
- File reading
- Basic error handling
- Demonstrates:
  - Line-based file I/O
  - Character-based file I/O
